== Installation ==

1. Upload the `nany-article-wp` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Ensure that **Yoast SEO** or **Rank Math** is installed and active (if using SEO meta features).
4. Enable **Application Passwords** in your WordPress settings (if using REST API features).
